Discord Bot für den FSR Info Server

Installation:
- Benötigt Node.js >=14
- Empfohlen: Visual Studio Code mit ESLint-Erweiterung (Auf Nachfrage die Verwendung von lokalem ESLint erlauben)
- Repository clonen
- `npm install`
- `src/token.ts` einfügen (siehe `src/token-template.ts`)
- `npm run build`
- `npm start`
